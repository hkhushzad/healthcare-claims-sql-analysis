## 🌟 **Project Overview**
This project analyzes **medical malpractice claims data** to uncover trends in **claim amounts, medical specialties, and the influence of private attorneys**.  
The goal is to provide **actionable insights** for healthcare organizations, insurance providers, and legal professionals.

## 👔 For Recruiters / Hiring Managers: How to Run the Query

1. **Open the project** in your SQL tool (PostgreSQL, DBeaver, or pgAdmin).  
2. **Open the SQL file** `medical_malpractice_claims_analysis.sql`.  
3. **Run the script** — it will:
   - Create the table for the dataset  
   - Import the CSV (`data/medicalmalpractice.csv`)  
   - Generate summary tables and insights  

> You don’t need to configure anything — just open and run the file to see the analysis.
